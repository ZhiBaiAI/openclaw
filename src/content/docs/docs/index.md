如果你第一次接触 OpenClaw，不建议一上来就翻命令和配置。
先按这条路线看，会更省时间：

1. 先看「7 步上手」里的第 1 步，弄清楚它适合帮你处理哪些事情。
2. 再跟着第 2 步完成安装，先把助手跑起来。
3. 接着看第 3 步到第 6 步，把它调成真正能配合你工作的状态。
4. 最后看第 7 步和参考资料，补上安全、维护和后续扩展。

## 这套文档适合谁

- 想在本地或云端搭一个可长期使用的 AI 助手
- 不想看太多术语，只想知道该怎么一步步做
- 已经装好 OpenClaw，但还没真正用顺手

## 你能学到什么

- 怎么选择适合自己的运行环境
- 怎么完成安装和首次对话
- 怎么把助手调成更符合你的使用习惯
- 怎么接入邮箱、日历、搜索和浏览器
- 怎么通过技能包扩展功能
- 怎么让它开始主动提醒、整理和执行任务
- 怎么更安全、更稳定地长期使用
